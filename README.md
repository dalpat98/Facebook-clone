# Facebook-clone
Dynamic Facebook login page created using HTML and CSS.
This also includes database connectivity using PHP used for login/register

The source code is composed of **HTML, CSS, JAVASCRIPT and PHP**.

- Login/register on local database
- Upload pics/posts to database after login
- Change profile picture
